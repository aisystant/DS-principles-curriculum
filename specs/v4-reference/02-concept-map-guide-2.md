# Заготовка: Руководство 2 «Методы саморазвития»

> **Объект:** Пилот — методы M001–M008.
> **Ось:** Мастерство (M1 — мастерство саморазвития).
> **Актуально со ступеней:** 2–5.
> **Время на подраздел:** 1,5 часа (7 подразделов = 10,5 часов/неделю).

---

## Основные идеи (из Pack)

1. **Методы = форма** (как я действую), не содержание (что я думаю). Руководство 2 — про форму, руководство 1 — про содержание.
2. **M001–M008** — 8 методов, каждый работает на Пилота, реализуется через IWE (приложение «Через IWE»).
3. **Степени мастерства** (CAT.003, FORM.006): Объяснение → Умение → Навык → Мастерство.
4. **Творческий конвейер** — где какой метод (PD.METHOD.019).
5. **Bottleneck-фокус** — не все сразу, а по узкому месту.
6. **Методы применяются к Пилоту**, IWE — контекст применения.

## Основные понятия (из Pack + текущих docs)

### Из Pack (обязательно в руководстве)
- Метод (M001–M008)
- Степень мастерства (Объяснение → Умение → Навык → Мастерство)
- Инвестированное / потраченное / потерянное время
- Ритуал входа в роль
- Бюджет времени
- Draft-конвейер
- PKM (personal knowledge management)
- Leisure-практики
- Каскад планирования
- cp.skl, bh.met, bh.stb, bh.inv

### Методы по номерам (PD.METHOD)
| Метод | Название | Что делает |
|-------|----------|------------|
| M001 | Учёт времени | Помодоро, трекер, slot_logged |
| M003 | Систематическое медленное чтение | Чтение для мышления, не для потребления |
| M004 | Мышление письмом | Внешнизация мыслей: заметка → черновик → заготовка |
| M005 | Мышление проговариванием | Проверка на «бытовой язык», прокладка нейронных связей |
| M006 | Организация досуга | Осознанное восстановление, эшелонированный досуг |
| M007 | Формирование окружения | Проектирование среды (физической и цифровой) |
| M008 | Стратегирование | Выбор направления: неудовлетворённости → намерения → стратегия |
| M009 | Планирование | Каскад: год → квартал → месяц → неделя → день |

### Из текущих docs (проверка — не забыть)
- **Вторая грамотность** — 8 практик как база XXI века
- **Творческий конвейер** — потребление → размышления → действие → досуг
- **Помидорка** — интервал фокусированной работы
- **Ритуал включения трекера** — роль, метод, рабочий продукт
- **Наградная помидорка** — лайфхак для преодоления сопротивления
- **Образовательная заметка** — результат медленного чтения + мышления письмом
- **Черновик / заготовка / публичный пост** — стадии мышления письмом
- **Эшелонированный досуг** — многоуровневая система отдыха
- **Ритмичность досуга / распорядок дня** — фундамент продуктивности
- **Бюджет времени** — распределение между постоянными/временными/важными/текущими/срочными
- **Формулировка работы** — кто, роль, метод, срок, ресурсы, рабочий продукт
- **Стадии обучения / степени мастерства** — Пропитка → Изучение → Применение → Обучение других

### Риски дублирования
- **Творческий конвейер** — в руководстве 1 как контекст, в руководстве 2 как инструмент.
- **Экзокортекс** — в руководстве 3 детально, здесь — как инструмент для методов.
- **ОРЗ** — в руководстве 3 как процесс, здесь — как применение к каждому методу.

## Чего НЕ должно быть в руководстве 2 (границы)

- Нет мировоззренческой дуги (это руководство 1)
- Нет архитектуры IWE (это руководство 3)
- Нет системного разбора (это руководство 4)
- Нет курсовых элементов (домашки, защита) — это в персональное руководство
