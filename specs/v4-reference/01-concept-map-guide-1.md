# Заготовка: Руководство 1 «Системное саморазвитие»

> **Объект:** Пилот (мировоззрение, роли, траектория, ценности).
> **Ось:** Мировоззрение (W).
> **Актуально со ступеней:** 1–5.
> **Время на подраздел:** 1 час (7 подразделов = 7 часов/неделю).

---

## Основные идеи (из Pack)

1. **Созидатель = Болид (Пилот + Машина)** — человек не отделён от инструментов; IWE — часть личности (FORM.089 §1).
2. **Мировоззрение = машинный код** — определяет, зачем учиться и куда идти. Методы = runtime, IWE = среда исполнения.
3. **Программа ЛР — фундамент для стиля жизни**, не курс. Читается в любом порядке.
4. **5 ролей созидателя** (FORM.087): Ученик → Интеллектуал → Профессионал → Исследователь → Просветитель.
5. **Мировоззренческая дуга** — 5 фаз: «Зачем учиться — непонятно» → «Я — система» → «Мир — система, и я в ней деятель» (FORM.080 §3.1).
6. **Ценности ≠ цели** — ценности = инвариант, машинный код. Цели = производные.
7. **Траектория развития** — 5 ступеней с bh/cp-порогами (FORM.080, FORM.089). Двойной gate: Аттестатор + Диагност.
8. **Творческий конвейер** — 4 стадии: потребление → размышления → действие → досуг (PD.METHOD.019).

## Основные понятия (из Pack + текущих docs)

### Из Pack (обязательно в руководстве)
- Болид, Пилот, Машина
- Мировоззренческая дуга (5 фаз)
- 5 ролей созидателя
- Ценности, намерения, неудовлетворённости
- Ступени ученика (5 ступеней)
- Двойной gate, bottleneck
- Творческий конвейер
- cp.wld, cp.rhy, bh.awr, bh.sys

### Из текущих docs (проверка — не забыть)
- **Системное саморазвитие** — проект выстраивания себя как системы
- **Системное мировосприятие** — стратегия на системном подходе
- **Неудовлетворённости** — движущая сила (есть в обоих руководствах)
- **Деятельностный кругозор** — широта взглядов
- **Экзокортекс** — «второй мозг» (пересекается с руководством 3)
- **Картина мира / мировоззрение** — совокупность фактов, знаний, убеждений
- **Агент, агентность, создатель** — ключевые роли
- **Калибр личности** — уровни охвата (FORM.090)
- **Мастерство** — главный капитал (vs информация)
- **Интеллект-стек / трансдисциплины** — фундамент мышления
- **SoTA** — лучшие теории
- **Квадрант компетенций** — 4 стадии освоения
- **Смысл жизни / сверхцель / счастье** — философский финал

### Риски дублирования
- **Экзокортекс** — в руководстве 1 кратко (ввод), детально в руководстве 3.
- **Интеллект-стек** — в руководстве 1 как контекст, в руководстве 4 как инструмент.
- **Агент** — в руководстве 1 как роль, в руководстве 4 как системное понятие.

## Чего НЕ должно быть в руководстве 1 (границы)

- Нет методов M001–M008 (это руководство 2)
- Нет инструкций по IWE (это руководство 3)
- Нет системного разбора как инструмента (это руководство 4)
- Нет курсовых элементов (домашки, недельный темп) — это в персональное руководство

## Проверка по текущим docs: что важного можно забыть?

| Понятие из docs | Статус | Где в новом руководстве |
|-----------------|--------|------------------------|
| Экзокортекс | ✅ есть в Pack | §Ввод, кратко |
| Неудовлетворённости | ✅ есть в Pack | §Ценности |
| Калибр личности | ✅ есть в Pack | §Ролевая траектория |
| Системное мировосприятие | ⚠️ неявно | §Ввод — объединить с «мировоззрение» |
| Деятельностный кругозор | ⚠️ неявно | §Ролевая траектория |
| SoTA | ⚠️ неявно | §Траектория развития |
| Квадрант компетенций | ❌ нет в Pack | Добавить? Ступени ученика = аналог |
| Смысл жизни / сверхцель | ⚠️ частично | §Ценности + §Траектория |

**Вывод:** Квадрант компетенций (4 стадии) из docs — полезная модель, но в Pack есть степени мастерства (Объяснение → Умение → Навык → Мастерство, FORM.006). Не дублировать, а сослаться.
