## Общие замечания по маппингу

1. **Курсовые элементы** (домашки, моделирование, защита, чек-листы) — не идут в универсальные руководства. Это в персональное руководство (Портной).
2. **Пересечения** — руководства ссылаются друг на друга. Каждое самодостаточно, но не изолировано.
3. **Порядок изучения** — в справочнике нет. Портной определяет порядок по RCS-профилю.
4. **Длина** — старые курсы ~10 недель каждый. Новые справочники — 2000–5000 слов каждый (читаются за 20–40 мин).
