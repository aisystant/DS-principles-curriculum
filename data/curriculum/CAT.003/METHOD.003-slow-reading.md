# CAT.003 — METHOD.003: Систематическое медленное чтение

**Ключевое различение:** медленное чтение ≠ скорочтение. «Медленно» = паузы для заметок в момент когнитивного резонанса. Работает только в паре с METHOD.004.

**Принцип культуры:** 25 минут ежедневно важнее 7 часов за 2 дня.

---

## Ритуал учёта времени (единый для всех слотов)

Каждый слот — осознанный ритуал:
1. **Стоп-момент** — пауза перед началом
2. **Три вопроса:** *Роль? Метод? РП?*
3. **Рабочее место** — стол, не кровать; один экран для чтения, другой для заметок
4. **Запуск трекера** — до начала
5. **Рефлексия 2 мин** — после: что зацепило?

---

## Степень 1 — Объяснение

*Цель: ≥5 ч инвестированного/нед, ≥1 слот в день*

**can-do:**
- Выбирает материал заранее, а не открывает ленту соцсетей
- При когнитивном резонансе останавливается и записывает заметку — не читает дальше
- Различает медленное чтение (паузы для заметок) и обычное (без остановок)

**Задание:**
Выбери одну книгу или статью (не ленту). Возьми слот 25 минут. Выполни ритуал, читай медленно. Когда почувствуешь «это важно» — остановись, запиши заметку. Цель: ≥3 образовательных заметки за слот.

**Assessment:**
≥5 образовательных заметок за неделю. Каждая заметка связана с конкретным понятием или принципом. Может объяснить, чем медленное чтение отличается от скорочтения.

---

## Степень 2 — Умение

*Цель: ≤1 пропуска/мес, до 2 ч/день, 2–3 мес подряд*

**can-do:**
- Ежедневная практика не нарушается — ритуал соблюдается даже в «плохой день»
- Еженедельно отслеживает систематичность по чек-листу
- Постепенно увеличивает длину слота: 25 → 45 → 60 мин для сложного материала

**Задание:**
Веди чек-лист медленного чтения каждый день 2 месяца. Отмечай: материал, длина слота, количество заметок. Раз в неделю — анализ: что помешало? Что улучшить?

**Assessment:**
≤1 пропуска/мес за 2 месяца. Заметки становятся более связными — видна динамика качества. Может описать свой метод работы с материалом.

---

## Степень 3 — Навык

*Цель: 2+ ч/день стабильно, пропуски — единичные исключения*

**can-do:**
- Практика приносит удовольствие — «не могу не читать медленно»
- Составляет собственные рекомендации по подбору материала и работе с когнитивным резонансом
- Мотивирует ближайшее окружение, делится заметками и заготовками

**Задание:**
Напиши заготовку с личными рекомендациями по медленному чтению для одного конкретного человека из окружения. Помоги ему настроить первый слот.

**Assessment:**
Окружение знает о практике. Может описать свой метод с примерами. Заметки стабильно превращаются в черновики (связь с METHOD.004).

---

## Степень 4 — Мастерство

*Цель: годы непрерывной практики. «Не понимаю, как можно жить без медленного чтения»*

**can-do:**
- Несколько учебников одновременно, двойной экран — стандарт работы
- Обучает других: проводит разбор, объясняет когнитивный резонанс и принцип заметок
- Медленное чтение питает стратегирование — материал из заметок становится источником гипотез

**Задание:**
Проведи сессию с кем-то из окружения: помоги выбрать материал, настроить рабочее место, взять первый слот. Разбери его заметки через неделю.

**Assessment:**
Годы непрерывной практики. Обучает других. Может показать связь: заметка → черновик → публикация.
