## Структура карточки

Каждая карточка содержит:
- **Ключевое различение** — главный концептуальный водораздел
- **Принцип культуры** — норма поведения в IWE
- **Ритуал учёта времени** — единый для всех практик (стоп-момент → Роль? Метод? РП? → трекер)
- **4 степени:** can-do + задание + assessment + количественные параметры

## Степени

| Степень | Название | Характеристика |
|---------|----------|---------------|
| 1 | Объяснение | Понимает основы, первые попытки |
| 2 | Умение | Регулярная практика несколько месяцев |
| 3 | Навык | Автоматизировано, приносит удовольствие |
| 4 | Мастерство | Годы практики, обучает других |
