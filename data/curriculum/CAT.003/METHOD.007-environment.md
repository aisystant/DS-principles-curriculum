# CAT.003 — METHOD.007: Формирование окружения

**Ключевое различение:** окружение — не данность, а результат творчества. Окружение воздействует впечатлениями: либо ускоряет развитие, либо приводит к деградации.

**Системные уровни:** личные вещи → рабочее место → семья → команда → сообщество → общество.

---

## Ритуал учёта времени (единый для всех слотов)

Каждый слот — осознанный ритуал:
1. **Стоп-момент** — пауза перед началом
2. **Три вопроса:** *Роль? Метод? РП?*
3. **Запуск трекера** — до начала

Для окружения: начинать с ближайшего уровня — рабочего места.

---

## Степень 1 — Объяснение

*Цель: 10–15 ч инвестированного/нед, 1 неделя*

**can-do:**
- Выделил ближайшее окружение вниманием — видит его как систему, а не фон
- Заполнил таблицу «Моё окружение»: физические объекты, люди, информационные каналы
- Написал учебный пост о своих неудовлетворённостях с окружением

**Задание:**
Заполни таблицу «Моё окружение» — перечисли всё, что тебя окружает (вещи, люди, каналы). Для каждого: впечатление (+/-). Напиши пост о том, что хочешь изменить.

**Assessment:**
Таблица заполнена. Может описать типы окружения (физическое, социальное, информационное). Пост опубликован.

---

## Степень 2 — Умение

*Цель: несколько мес, видны поэтапные изменения*

**can-do:**
- Создал проект «Формирование моего окружения» с конкретными гипотезами по каждому компоненту
- Нашёл «банку с огурцами» — сообщество людей с амбициозными целями (реальное или онлайн)
- Поэтапно меняет окружение: для каждого компонента — оставить / исключить / развивать

**Задание:**
Для каждого компонента таблицы «Моё окружение» напиши: что делать (оставить / исключить / развивать). Одну гипотезу протести за месяц. Запиши результат.

**Assessment:**
Видны конкретные изменения в окружении за несколько месяцев. Активное членство в одном сообществе с высокой миссией.

---

## Степень 3 — Навык

*Цель: систематически улучшает окружение, видны результаты*

**can-do:**
- Легко распознаёт объекты среды и их влияние — «вижу окружение»
- Формирует окружение проактивно: не реагирует, а создаёт
- Интегрирует практику в рабочие проекты и новые контексты

**Задание:**
Укажи 3 конкретных изменения в окружении за последний квартал. Для каждого: какой эффект дало?

**Assessment:**
Может указать конкретные изменения с измеримым эффектом. Практика применяется не только на личном уровне.

---

## Степень 4 — Мастерство

*Цель: «сегодняшнее окружение лучше вчерашнего» — годы. Работает на уровне команды и сообщества*

**can-do:**
- Осознанно меняет к лучшему всё вокруг — на уровне вещей, людей, сообщества
- Обучает других формированию окружения
- Переходит на высокие системные уровни: команда, предприятие, сообщество

**Задание:**
Помоги кому-то из окружения заполнить таблицу и сформулировать первую гипотезу изменения. Отслеживай вместе месяц.

**Assessment:**
Долгосрочная интеграция. Обучает других. Работает на уровне выше личного (команда / сообщество).
