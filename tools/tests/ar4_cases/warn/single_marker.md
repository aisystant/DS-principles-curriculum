# 1.22 Подраздел с одним маркером

В целом текст академический, без явных признаков ИИ-стилистики. Однако автор использовал
презентационный оборот: давайте разберёмся, что здесь происходит. Это типичный паттерн,
который должен вызвать WARN от линтера, но не FAIL (потому что net=1 ниже порога FAIL=3).

Дальше — обычный текст без маркеров. Никаких других проблемных фраз нет.
