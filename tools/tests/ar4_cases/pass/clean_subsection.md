# 1.20 Чистый подраздел

Подраздел в академическом стиле. Никаких маркеров ИИ-стилистики: нет «представьте себе»,
нет «давайте разберёмся», нет «золотых правил» и «в современном мире».

Описание метода: автор замеряет разрыв между намерением и действием через ежедневную
запись фактов. По итогам недели — пересмотр операционных условий.
