# 1.23 Реплики в кавычках исключаются

В примере из жизни человек говорит «давайте разберёмся, что я могу сделать сейчас» — это
речевой паттерн человека, не ИИ-стилистика. Реплики в «...» должны auto-исключаться,
поэтому AR.4 не должен ловить эту фразу как маркер.

Также pack-цитата `PD.FORM.087` в inline code или в блок-цитате > PD.D.142 — описание роли —
тоже исключается.
