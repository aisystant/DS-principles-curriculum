# 1.24 Текст с тремя параллельными конструкциями

Это касается не людей, не групп, не профессий.

Также не временно, не локально, не точечно.

И ещё раз: не день, не неделя, не месяц.

В тексте нет фраз из blacklist, но три тройные параллели на трёх разных строках
должны дать net=3 → FAIL.
