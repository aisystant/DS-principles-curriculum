# 1.31 Текст без маркеров

В этом подразделе автор пишет про систему, но не уточняет, что имеет в виду. Просто слово в тексте без типизации.

Дальше автор продолжает рассказ. Снова упоминается система, всё ещё без типизирующих фраз.

И ещё раз: система упоминается в третий раз. По-прежнему ни одного маркера различения нет рядом.

Линтер должен предложить добавить маркер различения для каждого из трёх упоминаний.
