# 1.32 Упоминание без контекстных слов

Автор пишет про целевую систему без указания нужных уточнений вокруг. Никаких слов из ожидаемого ряда в окружении нет.

Дальше автор рассуждает о других вещах: классификация систем, способы оценки, методы анализа. Но рядом с упоминанием целевой системы нужных уточнений нет.

Линтер должен предложить добавить пояснение к упоминанию.
